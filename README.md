# Crystallinity-Set Filtering (CSF) for HDPE/PP Blend Quantification

This repository contains a Python implementation of the **Crystallinity-Set Filtering (CSF)** method, a calibration-free approach for estimating the mass fraction of semicrystalline polymers in binary blends using DSC data.

The method is demonstrated for **HDPE/PP blends**, but can be extended to other semicrystalline polymer systems.

---

## Method Overview

The CSF method determines polymer composition without calibration curves by exploring a discretized set of possible crystallinity values and applying cristallinity constraints.

The mass fractions are defined as:

```
φ_HDPE = ΔH_m_HDPE / (ΔH_m⁰_HDPE · χc_HDPE)
φ_PP   = ΔH_m_PP   / (ΔH_m⁰_PP   · χc_PP)
```

Where:

- ΔH_m → melting enthalpy obtained from DSC of the polymer blend  
- ΔH_m⁰ → melting enthalpy of a 100% crystalline polymer (literature)  
- χc → degree of crystallinity  

---

## Crystallinity Set

The crystallinity set (C) is defined as a two-dimensional space where each point represents a possible pair of crystallinity values for both polymers:

```
C = { (x, y) | x = ε, 2ε, ..., 100 and y = ε, 2ε, ..., 100 }
```

---

## Filtering Strategy

The function **M(x, y)** is defined as the sum of the mass fractions calculated for each pair (x, y):

```
M(x, y) = φ_HDPE(x) + φ_PP(y)
```

Based on expected crystallinity ranges, the following boundary conditions are applied to filter the set:

1. χc_min < χc < χc_max  
2. χc_HDPE > χc_PP  
3. 1 − δ < M(x, y) < 1 + δ  

---

## Estimation

After applying the boundary conditions, the mass fractions are estimated by averaging all valid solutions:

```
φ_HDPE ≈ (1/n) Σ [ΔH_m_HDPE / (ΔH_m⁰_HDPE · x_i)]
φ_PP   ≈ (1/n) Σ [ΔH_m_PP   / (ΔH_m⁰_PP   · y_i)]
```

---

## Input Data

The script requires:

- ΔH_m_HDPE (J/g)  
- ΔH_m_PP (J/g)  
- ΔH_m⁰_HDPE (literature value)  
- ΔH_m⁰_PP (literature value)  

---

## Parameters

- χc_min, χc_max (crystallinity bounds)  
- ε (discretization step)  
- δ (filtering tolerance)
